# Aira 3D printer client

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8000 
npm run dev

# build for production with minification
npm run build
```

## Change configuration file

> src/config.js

## Clone and run
```bash
git clone https://github.com/vol4tim/ipfs-api-pubsub-ws
npm run local
```
